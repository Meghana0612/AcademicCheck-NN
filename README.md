# AcademicCheck NN

AI-powered plagiarism detection system using NLP and Transformer models.

## Features

- Semantic plagiarism detection
- Cross Encoder similarity scoring
- Explainable AI
- TF-IDF similarity
- Streamlit dashboard

## Tech Stack

- Python
- Streamlit
- spaCy
- Sentence Transformers
- Scikit-learn
- PyTorch

## Performance

- Accuracy: 84.27%
- F1 Score: 85.18%
- AUC: 92.01%
